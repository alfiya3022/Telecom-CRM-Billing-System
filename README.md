# Telecom CRM & Billing Management System

## 📌 Project Overview

This project simulates a **Telecom Customer Relationship Management (CRM) and Billing System** using **Microsoft SQL Server** and **Power BI**.

The objective was to:

* Design a normalized telecom database
* Enforce data integrity using constraints
* Automate business operations through stored procedures and triggers
* Perform business analysis using advanced SQL queries
* Visualize telecom KPIs through an interactive Power BI dashboard

---

## 🛠 Technologies Used

* Microsoft SQL Server
* T-SQL
* SQL Server Management Studio (SSMS)
* Power BI Desktop
* Git & GitHub

---

## 🗄 Database Design

The database consists of **10 interconnected tables**:

* Customers
* SIM Cards
* Circles
* Plans
* Subscriptions
* Usage Records
* Recharge History
* Bills
* Payments
* Support Tickets

### Key Features

* Primary Keys (PK)
* Foreign Keys (FK)
* Check Constraints
* Identity Columns
* Default Constraints
* One-to-Many Relationships

## 🔗 Constraints & Relationships

The database was designed with referential integrity using **Primary Key** and **Foreign Key** constraints.

### Example Relationships

* Customer → SIM Cards
* SIM Cards → Plans
* SIM Cards → Bills
* Bills → Payments
* Customers → Support Tickets
* SIM Cards → Usage Records

### Foreign Key Mapping

* Bills → SIM Cards
* Payments → Bills
* Recharge History → Plans
* Recharge History → SIM Cards
* Subscriptions → Plans
* Subscriptions → SIM Cards
* Support Tickets → Customers
* Support Tickets → SIM Cards
* Usage Records → SIM Cards
